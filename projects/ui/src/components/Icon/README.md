# Icon
> 通用Icon图标组件,可以作为单纯Icon,也可以直接作为IconButton

## 基本用法

传入图片url，作为Icon，传入onclick也可直接作为IconButton

```html
    <Icon :url="" :onclick="" className="" ></Icon>
```

## Props

| 字段 | 说明 | 类型 | 默认值
|----- | ----- | ----- | ----- 
| url | 图标的图片地址 | String | --
| onclick | 点击的回调函数 | Func | --
| className | 自定义className | String | --
| iconStyle | 图标的行内样式 | Object | --
